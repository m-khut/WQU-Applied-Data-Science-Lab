Project Overview
This project develops a **machine learning model to predict corporate bankruptcy** using real-world financial data from Polish companies. The focus is on handling messy, imbalanced financial datasets and building robust classification models that prioritize meaningful business outcomes.

---

Objective
Predict the likelihood of company bankruptcy using financial indicators, with emphasis on handling **imbalanced data** and optimizing **precision and recall**.

---

Skills & Concepts Covered
- Working with compressed JSON financial data  
- Data wrangling and feature preparation  
- Handling severely imbalanced datasets  
- Classification modeling  
- Model evaluation using precision, recall, and confusion matrices  
- Model persistence and deployment-ready workflows
