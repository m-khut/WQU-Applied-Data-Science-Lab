This project explores **real-world housing data from Buenos Aires** and applies end-to-end data science workflow skills to **build predictive models for property prices**.

It focuses on learning how to:
 - Clean and prepare housing datasets for modeling
 - Explore relationships between features and price
 - Build and evaluate predictive models
 - Apply lessons in a final assignment-style notebook workflow

Objective
 - Build models that predict **property sale prices** using available listing features, starting with the most intuitive driver:
