Project Overview:
This project analyzes 21,000+ residential property listings in Mexico sourced from Properati.com to answer a core real-estate and data-science question:
  Are housing prices influenced more by property size or by location?
Using structured data analysis and exploratory techniques, the project investigates how these two factors relate to sale prices and which has a stronger impact.

Objective
  Evaluate the relationship between property size and sale price
  Evaluate the relationship between location and sale price
  Compare both effects to determine which factor is more influential

Key Concepts & Skills Applied
  Data organization using Python data structures
  Data cleaning and preparation with pandas
  Exploratory Data Analysis (EDA)
  Data visualization:
    Scatter plots
    Box plots
  Statistical analysis:
  Correlation analysis
  Analytical reasoning and interpretation of results

Dataset
  Source: Properati.com (real estate listings)
  Scope: ~21,000 properties across Mexico
  Key Features Used:
    Sale price
    Property size (area)
    Geographic location
